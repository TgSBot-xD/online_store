# SneakMax — интернет-магазин кроссовок

Одностраничное веб-приложение интернет-магазина кроссовок. Проект реализован по макету из Figma и включает каталог товаров с фильтрацией, корзину покупок, оформление заказа и интерактивный подбор обуви.

**[Demo](https://sneakemax.netlify.app/)** | **[Figma-макет](https://www.figma.com/file/EEiUY5PZ7GiO2hYRth11Yy/SneakMax-YouTube?type=design&node-id=43-389&mode=design&t=IWJYyEEC3aKvHbTQ-0)**

## Стек технологий

- **React 18** + **TypeScript** — UI-фреймворк и типизация
- **Redux Toolkit** — управление глобальным состоянием
- **styled-components** — CSS-in-JS стилизация
- **React Hook Form** — работа с формами и валидация
- **Axios** — HTTP-запросы к API
- **React Slick** — карусели изображений
- **React Slider** — ползунок диапазона цен
- **React Yandex Maps** — интеграция Яндекс.Карт
- **React Simple Star Rating** — рейтинг товаров

## Функциональность

- Каталог кроссовок с загрузкой данных из внешнего API
- Фильтрация товаров по цене, полу и размеру
- Корзина покупок с подсчётом итоговой стоимости
- Оформление заказа через модальное окно
- Пошаговый подбор обуви (тип, размер, пожелания)
- Раздел FAQ с раскрывающимися вопросами
- Форма обратной связи
- Контакты с интеграцией Яндекс.Карт
- Пагинация каталога

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/<username>/online-store.git
cd online-store

# Установить зависимости
yarn install

# Запустить dev-сервер
yarn start
```

Приложение будет доступно по адресу `http://localhost:3000`.

### Переменные окружения

При необходимости можно переопределить URL API — скопируйте `.env.example` в `.env` и измените значение:

```bash
cp .env.example .env
```

### Сборка

```bash
yarn build
```

## Структура проекта

```
src/
├── api/                  # Запросы к API
├── assets/               # Шрифты, изображения, SVG-иконки
├── components/           # Переиспользуемые компоненты
├── global/styles/        # Глобальные стили и подключение шрифтов
├── helpers/              # Утилиты (форматирование чисел и др.)
├── hook/                 # Кастомные хуки (типизированные Redux-хуки)
├── modules/              # Функциональные модули приложения
│   ├── Catalog/          #   Каталог товаров с фильтрацией
│   ├── ConnectUs/        #   Форма обратной связи
│   ├── Contacts/         #   Контакты и Яндекс.Карты
│   ├── Header/           #   Шапка, навбар, корзина, оформление заказа
│   ├── Questions/        #   FAQ-секция
│   ├── SelectionOfShoes/ #   Пошаговый подбор обуви
│   └── Team.tsx          #   Команда
├── pages/                # Страницы (HomePage)
├── store/                # Redux store и редьюсеры
├── types/                # TypeScript-декларации
└── UI/                   # UI-Kit (Button, Input, Text, BlockText и др.)
```
